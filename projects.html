Voice Query Language [Mangohacks 2019]

Posted on 3/17/2019

Ask google to give you common business database information without every touching SQL. Created a flask API in a virtual environment. The flask API interfaces between the business 8 base database and the voice assistant API. The flask API includes a Natural Language Processing model along with several functions to handle query communication with the google cloud APIs and the database. NLP was built using sklearn, spacey, and pandas. Deployed the flask API on Heroku and managed version control through GIT.
